# Sample Assets

This directory contains local image assets used for testing and detector tuning.

- `BMP/`
  - Bitmap x-ray scans used as standalone image inputs.
- `TIF/`
  - TIFF x-ray scans used as standalone image inputs.

Notes:

- Bundled sample DICOM artifacts have been removed from the repository.
- The assets in this directory are ordinary image files, not DICOM studies.
